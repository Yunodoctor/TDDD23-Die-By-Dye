# TDDD23_Die_By_Dye
Design and Programming of Computer Games

Die By Dye is a 2D top-down shooter game with locked view. The goal of the game is to defeat enemies by shooting them with weapons. To shoot an enemy, the colour of the weapon have to match the colour of the enemy. The game was developed using the game engine Unity with the game assets coded in C#. All the art and animations was created in Photoshop and Illustrator.
